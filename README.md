# EdRegistry

EdRegistry is a **website explorer for EdFile data from LEGO TTGames**.  
It allows you to **view, search, sort, and filter** a CSV containing information extracted from EdFiles of LEGO TTGames games using a custom parser.

The site is fully **static**, hosted on **GitHub Pages**, and features:

- Interactive table with **sorting, pagination, and column filters**
- **Instant search**
- Modern **Dark / Light mode**, responsive
