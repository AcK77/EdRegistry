EdRegistry

EdRegistry is a dynamic, interactive registry of EdFiles used in LEGO TTGames titles. The data displayed here is generated from GitHub sources and curated by JayFrancoe using a custom serializer. Explore file types, categories, and structures with search and filtering tools.
